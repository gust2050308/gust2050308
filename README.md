## `MOD_03` DEPLOYMENTS

### `PROJECT_001` — Orbis · `Deployed (demo)`

**Plataforma de reservas turísticas**
`Next.js` `TypeScript` `Supabase` `Stripe` `Google Maps API`

Orbis permite a los usuarios explorar destinos, reservar excursiones y pagar en línea, mientras los administradores gestionan compras, reembolsos y métricas operativas desde un dashboard propio.

- Autenticación y protección de rutas por roles
- Pagos con Stripe Checkout + webhooks
- Dashboard admin con KPIs e ingresos mensuales
- CRUD de destinos con mapas y carga de imágenes
- Tablas interactivas (TanStack Table)

### `PROJECT_002` — IL Stockroom · `Internal / Active`

**Sistema de gestión de inventario — International Label**
`Next.js (App Router)` `TypeScript` `Supabase (PostgreSQL)` `Tailwind` `shadcn/ui`

Sistema interno para digitalizar y centralizar el control de inventario de una empresa de impresión y etiquetado, construido durante mis prácticas profesionales.

- Catálogo de insumos, proveedores (con mapas) y existencias
- Órdenes de compra con generación automática de PDF
- Requisiciones de producción con lógica de negocio propia
- Chat interno en tiempo real + alertas automáticas por correo
- Asistente con IA (Groq/Llama) integrado

**Backends independientes (auth + persistencia)**
`Java 17` `Spring Boot 3` `Spring Security` `JWT` `MongoDB` `PostgreSQL` `MapStruct` `Swagger`

Dos APIs REST construidas desde cero para reforzar mis fundamentos de backend en Java, explorando autenticación segura, persistencia SQL/NoSQL y buenas prácticas de arquitectura.
